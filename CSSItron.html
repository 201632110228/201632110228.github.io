CSS overview

•Cascading Style Sheets (CSS).

•How does a style definition display HTML elements.

•Styles are usually stored in stylesheets.

•Adding styles to HTML 4.0 is to solve the problem of separation of content and performance.

•External style sheets can greatly improve productivity.

•External stylesheets are typically stored in CSS files.


Style solves a common problem

•HTML tags are originally designed to define document content.By using tags such as <h1 >, <p >, <table >, HTML is meant to be "this is the title", "this is the paragraph", "this is the table" and the like.The document layout is also done by the browser without any formatting tags.

•Due to two main browser (Internet Explorer and Netscape) keep new HTML tags and attributes (such as font tags and color attributes) added to the HTML specification, create a document content clearly independent of document presentation layer sites become more and more difficult.

•To solve this problem, the world wide web consortium (W3C), the nonprofit standardization alliance, has taken on the task of standardizing HTML and creating Style (Style) outside of HTML 4.0.

•All of the major browsers support cascading style sheets.


Style sheets greatly improve work efficiency

•The stylesheet defines how to display HTML elements, like the font tags and color attributes of HTML 3.2.Styles are usually kept in an external.css file.By simply editing a simple CSS document, an external style sheet enables you to change the layout and appearance of all the pages in your site at the same time.

•CSS can be a breakthrough in WEB design because it allows you to control multiple page styles and layouts.As a web developer, you can define styles for each HTML element and apply it to any number of pages you want.If a global update is required, simply change the style, and all the elements in the site will be automatically updated.


Multiple styles will cascade to one

•Style sheets allow you to specify style information in a variety of ways.Styles can be specified in a single HTML element, in the header element of an HTML page, or in an external CSS file.You can even refer to multiple external style sheets within the same HTML document.

What style is used when the same HTML element is defined by more than one style?

In general, all styles are stacked in a new virtual style sheet based on the following rules, with the number 4 having the highest priority.

•The browser defaults

•External style sheet

•Internal style sheet (located inside the <head> tag)

•Inline style (inside HTML elements)


So, an inline style (inside an HTML element) has the highest priority, which means that it will take precedence over style of the following statement: the < head > tag, the declaration of a style style declarations in the external style sheet, or in the browser style statement (the default).
