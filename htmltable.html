    This example demonstrates various types of table borders

    Form

    The table is defined by the <table> label.Each table has several rows (defined by the <tr> tag), and each row is split into several cells (defined by the <td> tag).The letter td refers to table data, the content of a data cell.A data cell can contain text, images, lists, paragraphs, forms, horizontal lines, tables, and so on.

    <table border="1">

    <tr>

    <td>row 1, cell 1</td>

    <td>row 1, cell 2</td>

    </tr>

    <tr>

    <td>row 2, cell 1</td>

    <td>row 2, cell 1</td>

    </tr>

    </table>

    The browser displays the following:


    Row 1, cell 1 row 2, cell 2

    Row 2, cell 1 row 1, cell 2

    Table and border attributes

    If the border property is not defined, the table will not display the border.Sometimes it's useful, but most of the time, we want to display borders.

    Use the border property to display a table with borders:



    <table border="1">

    <tr>

    <td>Row 1, cell 1</td>

    <td>Row 1, cell 2</td>

    </tr>

    </table>



    Table header

    The table header USES the <th> label to define.

    Most browsers display the header as text in bold text


    <table border="1">

    <tr>

    <th>Heading <th>

    <th>Another Heading <th>

    <td>row 2, cell 1</td>

    <td>row 2, cell 1</td>

    </tr>

    <tr>

    <td>row 2, cell 1</td>

    <td>row 2, cell 1</td>

    </tr>

    </table>

    The browser displays the following:


    Heading   Anthor Heading

    Row 1, cell 1 row 2, cell 2

    Row 2, cell 1 row 1, cell 2
