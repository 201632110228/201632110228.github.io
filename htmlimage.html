    Image tags (< img >) and source properties (Src)

    In HTML, the image is defined by the tag

    <img> is an empty label, meaning that it contains only attributes and no closing tags

    To display an image on the page, you need to use the source attribute (SRC).The SRC refers to the "source".The value of the source property is the URL of the image.

    The syntax for defining images is:

    <img src="url" />

    The URL refers to where the image is stored.If called "boat. GIF images, the images directory at www.w3school.com.cn, then the URL to http://www.w3school.com.cn/images/boat.gif.

    The browser displays the image in the document where the image tag appears.If you place the image tag between two paragraphs, the browser will first display the first paragraph, then display the image and finally show the second paragraph.

    -----------------------------------------------------------------------------------------------------

    Replace text attributes (Alt)

    The Alt attribute is used to define a set of ready alternate text for the image.The value of the replacement text attribute is user-defined.

    <img src="boat.gif" alt="Big Boat">

    When the browser cannot load the image, replace the text attribute to tell the reader what they lost. At this point, the browser will display the alternative text instead of the image. Adding alternate text attributes to the images on the page is a good habit, which helps to display information better and is very useful for those who use plain text browsers.

    -----------------------------------------------------------------------------------------------------

    Basic precautions - useful tips:

    If an HTML file contains ten images, you need to load 11 files to display the page correctly. Loading pictures takes time, so our advice is: use pictures with caution.
