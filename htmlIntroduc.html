    • What is HTML?

    •HTML is a language used to describe a web page.

    •HTML is a language used to describe a web page.

    •HTML is not a programming language, but a markup language.

    •Markup language is a set of tag tags (markup tag).

    •HTML USES tag tags to describe web pages.

    HTML tags

    •HTML tags are often referred to as HTML tags.

    •The HTML tag is the keyword surrounded by the brackets, such as < HTML >.

    •HTML tags usually come in pairs, such as <b>and </b> .

    •The first TAB in the tag pair is the start tag, and the second tag is the end tag.

    •The start and end tags are also known as open labels and closed tags.

    HTML document = web page

    •HTML document description page.

    •HTML documents contain HTML tags and plain text.

    •HTML documents are also called web pages.
