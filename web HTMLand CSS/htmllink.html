    HTML hyperlink (link)

    •A hyperlink can be a word, a word, or a set of words, or an image that you can click on to jump to a new document or part of the current document.

    •When you move your mouse pointer over a link in the web page, the arrow will turn into a small hand.

    •We create links in HTML by using the <a> tag.

    •There are two ways to use the <a> tag:

    1:By using the href attribute - create a link to another document

    2:Use the name attribute - create a bookmark in the document

    HTML link syntax

    The HTML code for the link is simple.It looks something like this:

    <a href="https://201632110228.github.io/">WEB FACTORY</a>

    The href attribute specifies the target of the link.

    The text between the start tag and the end tag is displayed as a hyperlink.

    -----------------------------------------------------------------------------------------------------

    HTML link - name attribute

    You can use the name attribute to create a bookmark in an HTML page.

    Bookmarks are not displayed in any special way and are not visible to the reader.

    When using named anchors (named anchors), we can create a direct jump to the named anchor links (for example, a section of the page), so that users do not need to constantly scroll to find the information they need.

    <a name="label"> anchor (display text on the page)</a>

    •Tip: the name of the anchor can be any name you like

    •Tip: you can use an id attribute to replace the name attribute, which is also valid

    -----------------------------------------------------------------------------------------------------

    Example



    First, we name the anchor (create a bookmark) in an HTML document:

    <a name="tips">basic precautions - useful tips </a>

    Then, we create a link to that anchor in the same document:

    <a href="#tips">useful reminder </a>

    You can also create links to that anchor on other pages:

    <a href = "http://www.w3school.com.cn/html/html_links.asp#tips" > helpful hints </a>

    In the code above, we add the # symbol and anchor name to the end of the URL, and you can directly link to the naming anchor for tips.
