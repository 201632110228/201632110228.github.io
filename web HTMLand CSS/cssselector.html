CSS element selector

The most common CSS selector is the element selector. In other words, the element of the document is the basic selector. If you set the HTML style, the selector will usually be an HTML element such as p, h1, em, a, or even HTML itself:

html {color:black;}

h1 {color:blue;}

h2 {color:silver;}

    • You can switch one style from one element to another

    Suppose you decide to set the paragraph text above (instead of the h1 element) to grey.Just change the h1 selector to p:

    html {color:black;}

    p {color:gray;}

    h2 {color:silver;}

    • Type selector

    In the W3C standard, the element selector is also known as type selector (type selector).

    "Type selector matches the name of the document language element type.Type selector matches every instance of this element type in the document tree.

    The following rules match all h1 elements in the document tree:

    h1 {font-family: sans-serif;}
